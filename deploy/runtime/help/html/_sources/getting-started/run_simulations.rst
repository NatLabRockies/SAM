Run Simulation
==============

After reviewing and modifying inputs on the :doc:`input pages <input_pages>`, click **Simulate** to run a simulation. 

After you run a simulation, SAM may display :doc:`notices <../results/notices>` with information about the simulation. You can also display the notices any time by clicking **Notices**.

.. image:: ../images/SS_MainWindow-RunAllSimulationsButton.png
   :align: center
   :alt: SS_MainWindow-RunAllSimulationsButton.png

You can also run a simulation from the Case menu or by pressing the F5 key:

.. image:: ../images/SS_MainWindow-MenuSimulate.png
   :align: center
   :alt: SS_MainWindow-MenuSimulate.png
