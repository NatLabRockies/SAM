# System Advisor Model (SAM)

This is the gh-pages branch for NLR's System Advisor Model (SAM). The main SAM code respository is at https://github.com/NatLabRockies/SAM/tree/develop.

The following URL points to the release notes: https://natlabrockies.github.io/SAM/doc/releasenotes.html.

For now gh-pages stores the release notes that are displayed on the SAM website at https://sam.nlr.gov/download and from the SAM application's Help window.

Eventually, we may consider deploying SAM's Help system from this repository.
