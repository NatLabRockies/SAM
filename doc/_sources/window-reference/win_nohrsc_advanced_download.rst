Advanced NOHRSC Download Window
===============================

The Advanced NOHRSC Download window provides access to online databases of snow depth data for the detailed photovoltaic model's snow loss model. To open this window, enable snow losses on the :doc:`../detailed-photovoltaic-model/pv_soiling_shading_snow`, choose the **Enter or download time series snow data** option, and click **Download Snow Data**.

For information about the snow loss model, see :ref:`pvsnowlosses`.

To download snow depth data:

#. Type a location name or geographic coordinates for the array's location. Be sure to use the same location as the one on the Location and Resource page.

.. note:: SAM does not update the snow depth data when you change the weather file on the :doc:`../detailed-photovoltaic-model/pv_location_and_resource` page. Be sure to download snow depth data each time you change the weather file.

#. Click **Find** to generate a list of datasets with snow data for the location.

#. Choose a dataset and year from the list. (Click the name of a dataset to expand the list of years.)

#. Click **Download Snow Depth Array** to download the data.

When the download finishes, the window closes. You can click **Edit array** on the Soiling Shading Snow page to see the snow depth data.
