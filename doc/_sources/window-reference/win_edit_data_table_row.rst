Edit Data Table by Row
======================

For some inputs, such as costs for the Wind model's Offshore Balance of System Cost model, the input is table of variables rather than a single value. For example, the General input is a cost category of several variables:

.. image:: ../images/SS_EditTable.png
   :align: center
   :alt: SS_EditTable.png

To enter a value for one of the variables, type its value in the Value column.
