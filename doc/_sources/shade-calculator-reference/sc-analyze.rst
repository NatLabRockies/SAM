Analyze
=======

Click **Analyze** after creating the shading scene to generate shading data. You can then export shading data for use in simulation models and other software. This section is a quick reference for the Analyze options. For more details see:

* :doc:`Generate shade data <../shade-calculator-howto/sc-generate-data>`


* :doc:`Export shade data <../shade-calculator-howto/sc-export>`


.. note:: To generate SAM shading inputs, click :doc:`Save and Close <sc-save-close>`. You do not need to click **Analyze** to generate shading data for SAM.

**Diurnal analysis**
  Click this button to a generate table of hour-by-month beam shade loss percentages.

  When your scene contains more than one active surface, you can generate a separate hour-by-month table for the surfaces by assigning group names to the surfaces. See :doc:`Generate shading data for details <../shade-calculator-howto/sc-generate-data>`  .

**Time series analysis**
  Generate a CSV file of beam shade loss percentages for each of the time steps in a year.

**Diffuse analysis**
  Generate a sky diffuse loss percentage.
