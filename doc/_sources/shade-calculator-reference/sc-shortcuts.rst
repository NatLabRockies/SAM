Shortcuts
=========

The following keyboard and mouse shortcuts make it easier to zoom and pan the scene.

. The scene must have focus for these shortcuts to work. Be sure to click somewhere in the scene before using the shortcuts.

Open and save files
...................

**Ctrl-O**
  open file

**Ctrl-S**
  save file

Zoom and pan the scene
......................

**Arrow keys (left, right, up, down)**
  move the scene

**I**
  zoom in (make objects bigger)

**O**
  zoom out

**C**
  center the scene

**Z**
  change to elevations view

**T**
  change to bird's eye view

**S**
  change to 3D scene view

**drag (mouse)**
  rotate scene (3D scene view only)

**Shift-drag**
  pan scene

**Ctrl-drag**
  zoom scene
