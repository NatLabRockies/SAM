Hybrid Systems
==============

SAM's hybrid system models combine two or more power generation subsystems with battery storage. SAM runs a simulation for each subsystem, and adds the subsystem generation profiles together to calculate hybrid system's generation profile. It then dispatches the battery based on the hybrid generation profile.

The available hybrid system configurations are:

* PVWatts Wind Battery

* PVWatts Wind Fuel Cell Battery

* Photovoltaic Wind Battery 

* Custom Generation PVWatts Wind Fuel Cell Battery
