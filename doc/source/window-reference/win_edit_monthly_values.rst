Edit Monthly Data
=================

Some input variables, such as soiling losses, require a value for each month of the year.

.. image:: ../images/SS_MonthyData-Table.png
   :align: center
   :alt: SS_MonthyData-Table.png

Type a value in the table for each month.

To assign the same value to all months, type a value for **Enter single value** and click **Apply**.

**Copy**
  Copy the 12 values from the table to your computer's clipboard.

**Paste**
  Paste values from your computer's clipboard to the table.

**Enter single value**
  Type a value to apply to all months.

**Apply**
  Apply the value in Enter single value to all months.
