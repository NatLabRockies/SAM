Rotate objects
==============

You can rotate rectangular objects in the horizontal plane. In bird's eye view, drag the rotation handle, or in any view change the rotation property.

For active surfaces, the rotation angle is its azimuth angle, with 180 degrees defined as the rotation with the surface's width parallel to the X axis.

To rotate an object with your mouse.

#. Click **Bird's eye**.

#. Click the object to select it.

#. Drag the rotation handle |ss-icon-rotate|.

To rotate an object by editing its properties

#. In any view, click the object to select it.

#. Change the value of the rotation property and press the Tab or Enter key.
