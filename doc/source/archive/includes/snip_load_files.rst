
You can use one of the load data files supplied with SAM, or prepare your own load data file. 

A load data file is a text file with 8,761 rows: The first row is a text header that SAM ignores, and the remaining 8,760 rows must contain average hourly electric demand data in kilowatts. The first data element represents the hour beginning at midnight and ending at 1 a.m. on January 1.

To import load data from a properly formatted text file:

#. On the Electric Load page, click **Edit**.

#. In the Edit Hourly Data window, click **Import**.

#. Navigate to the folder containing the load data file and open the file. To use U.S. load data supplied with SAM, navigate to the *\samples\Residential Load Data* folder.

SAM displays the data in the data table. Use the scroll bars to see all of the data.

#. Click **OK** to return to the Electric Load page.

To import load data from a spreadsheet or other file:

#. On the Electric Load page, click **Edit**.

#. Open the spreadsheet containing the load data. The data must be in a single column of 8,760 rows, and expressed in kilowatts.

#. In the spreadsheet, select the load data and copy it.

#. In the Edit Hourly Data window, click **Paste**. 

#. You can also copy data from the Edit Hourly Data window by clicking **Copy**, or export the data to a text file by clicking **Save**.
