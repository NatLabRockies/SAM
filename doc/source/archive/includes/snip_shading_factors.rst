
To define the timing of array shading:

1. In the Number of Categories list, select a number between 1 and 9.

2. In the Shading Categories list, assign a shading factor value between 0 and 1 to each shading factor category.

3. In the shading matrix, select time periods for the first category and type the letter 'A.'

4. Repeat step 3 for each of the remaining categories, typing the appropriate letter.

5. Click OK to finish.

For more information and detailed instructions, press the F1 key or click the Help button.
