Main Window
===========

After you :doc:`choose the models for your analysis <../getting-started/choose_models>`, SAM displays the main window, which gives you access to the input pages for each case in the project:

.. image:: ../images/SS_MainWindow-Inputs.png
   :align: center
   :alt: SS_MainWindow-Inputs.png

Shortcut Menus
~~~~~~~~~~~~~~

A |SS_Button-ContextArrow| down arrow in SAM indicates a context menu. Click the button to show a list of commands or options:

.. image:: ../images/SS_ShortcutMenu.png
   :align: center
   :alt: SS_ShortcutMenu.png

Case Tabs
~~~~~~~~~

The case tab cases in the project and a context menu with commands for :doc:`managing cases <../reference/manage_cases>`. A project may consist of a single case, or may contain more than one case.
